# gradesAnalyzer-OpenMP

Na pasta do projeto, estão disponíveis os seguintes comandos:

* Compilar e rodar o código sequencial
```
    make runseq
```

* Compilar e rodar o código paralelo
```
    make runpar
```

* Compilar ambos programas
```
    make all
```
